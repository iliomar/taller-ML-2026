# Introducción a Machine Learning para Docentes de Escuela Superior 2026

Workshop práctico de cinco días diseñado para introducir conceptos fundamentales de **Machine Learning** a docentes de escuela superior, con enfoque en aplicaciones educativas y proyectos implementables en el salón de clases.

---

## Objetivo del Workshop

Brindar a docentes las herramientas conceptuales y prácticas necesarias para:

- Comprender qué es Machine Learning.
- Trabajar con datos utilizando Python.
- Entrenar y evaluar modelos básicos.
- Diseñar actividades y proyectos relacionados con análisis de datos.
- Discutir aspectos éticos y de sesgo en modelos predictivos.
- Integrar pensamiento basado en datos en distintas disciplinas académicas.

Este workshop no requiere experiencia avanzada en programación.

---

## Estructura del Workshop (5 días)

### Día 1 — Python y Fundamentos de Datos
- Repaso de Python.
- Trabajo con tablas de datos (pandas).
- Visualización básica.
- Discusión sobre patrones en datos.

### Día 2 — Introducción a Machine Learning
- ¿Qué es Machine Learning?
- Features y labels.
- Train/test split.
- Métricas básicas de evaluación.

### Día 3 — Entrenamiento y Overfitting
- Uso de un dataset real (Kaggle – SDSS17).
- Modelos básicos (Logistic Regression, Decision Tree).
- Validación y comparación de modelos.
- Demostración de overfitting.

### Día 4 — Machine Learning en el Salón de Clases
- Diseño de proyectos educativos.
- Ética y sesgo en modelos.
- Adaptación del dataset astronómico a nivel escolar.
- Desarrollo de una plantilla de proyecto estudiantil.

### Día 5 — Mini Capstone y Reflexión
- Proyecto en grupos.
- Presentaciones cortas.
- Discusión sobre implementación futura.
- Recursos adicionales.

## Dataset Utilizado

Para las sesiones prácticas utilizamos:

**Stellar Classification Dataset – SDSS17**  
Fuente: Kaggle  
https://www.kaggle.com/datasets/fedesoriano/stellar-classification-dataset-sdss17

Este dataset contiene mediciones astronómicas reales que permiten clasificar objetos como:
- STAR
- GALAXY
- QSO

Se utiliza con fines educativos para demostrar clasificación supervisada.
